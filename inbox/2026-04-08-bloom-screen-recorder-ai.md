# Bloom — Open Source Screen Recording for AI Workflows

Andrew shared this with the note: "Interesting screen recording/AI play here"

**Raw link:** https://github.com/video-db/bloom

## What It Is

Open-source Loom alternative. Record → Query → Automate. Local-first screen recording with AI indexing — captures screen, mic, and system audio locally, then uploads to VideoDB for transcription, visual embeddings, and AI agent access via API.

## Key Details
- **Tech:** Electron 39.0, Node.js 18+, SQLite (local), VideoDB SDK (cloud processing)
- **Status:** Active — 136 stars, 26 forks, latest release v2.1.0 (March 2026). macOS + Windows installers available.
- **Philosophy:** "Your recorder should not trap your data." Local ownership, no vendor lock-in, open-source UI layer.
- **AI angle:** Designed as a foundation for building agentic systems on top of recordings. Claude Code integration mentioned.
