# Claude Managed Agents — Business Guide (Non-Technical)

Source: Article/newsletter from "Return My Time" community. Positioned as a guide for non-technical people to sell AI agent services using Claude Managed Agents.

## Key New Info (not already in DOC-025/DOC-007)

### Pricing (confirmed)
- Standard Claude API token rates
- $0.08 per session-hour for active runtime
- Web search: $10 per 1,000 searches
- Typical 10-min session = a few cents
- Heavy usage stays under $100/month for most use cases

### The 4 Building Blocks
1. **Agent** — instructions, model, tools (the "job description")
2. **Environment** — pre-loaded workspace with software/tools ("new employee's laptop")
3. **Session** — running conversation, persistent memory, hours-long
4. **Events** — messages back and forth, status updates, approval requests

### Permission System (sellable to risk-averse clients)
- **Auto-run** — agent handles everything (good for internal tools)
- **Approval required** — agent pauses before acting (good for client-facing)
- Can mix: auto for reads/searches, approval for sends/updates
- Key sales line: "The agent will draft the email, but it won't send anything without your approval."

### AI Services Business Playbook
1. Run a $999 AI audit to identify client's biggest time wasters
2. Build a managed agent that solves their top problem
3. Deploy on Anthropic's infrastructure (they handle security, uptime)
4. Charge $500/month to maintain and improve
5. 4 clients = $2,000/mo recurring. 10 clients = $5,000/mo.

### Use Cases as Service ($1,500-5,000 setup + $500/mo)
- Client onboarding agents (collect docs, send reminders, set up accounts)
- Report generators (multi-source data → executive summaries)
- Customer support agents (answer common Qs, escalate edge cases, log to CRM)
- Document processors (extract data from raw files — finance teams using this)
- Project management agents (join Asana/Linear, pick up tasks)

### Use Cases for Own Business
- Email reader/responder in your voice
- Competitor monitoring → weekly briefings
- Content agent: rough notes → finished blog posts, social copy, newsletters

### Target Market
Law firms, accounting practices, real estate agencies, medical offices — they'll never set this up themselves. They want someone to do it for them.
