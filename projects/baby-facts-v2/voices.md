# Voice Profiles — Baby Facts V2

Six voices total: 3 per character. Designed to rotate within each character's pool so viewers don't pattern-match the same voice to the same face too quickly. All variants stay inside the character's tonal range — think "same person in different moods" rather than three distinct people.

Paste each `Description` into ElevenLabs Voice Design. Use the `Sample text` for the preview generation so you're evaluating against real Baby Facts content, not generic script lines.

Once generated, paste the ElevenLabs voice ID into the `Voice ID` slot below, and also into the character spec file.
