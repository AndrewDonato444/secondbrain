# Manager Forge

## Project Overview
- **Project Name:** Manager Forge
- **Description:** Training program equipping first-time SaaS sales managers (0–12 months in role) with practical tools for emotional leadership, information filtering when managing upward, and sustainable operating cadence.
- **Target Audience:** First-time sales managers in SaaS/software companies
- **Status:** Content development in progress (3 of 12 core modules completed)

## Task Folders
- **Curriculum-Design** — Module creation (scripts, worksheets, assessments, role-plays)
- **Market-Research** — Competitive analysis, customer discovery, pricing research
