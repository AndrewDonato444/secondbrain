# Manager Forge

**Training program equipping first-time SaaS sales managers (0–12 months in role) with practical tools for emotional leadership, information filtering when managing upward, and sustainable operating cadence.**

- **STATUS.md** — Real-time progress tracker
- **SYSTEM.md** — Project system prompt
- **tasks/** — Task folders (Curriculum-Design, Market-Research)
- **docs/** — Research findings, market analysis summaries
