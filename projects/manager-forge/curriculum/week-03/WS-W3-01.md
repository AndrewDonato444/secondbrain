## What This Is

A single-rep coaching plan for an AI-related performance issue. You pick one rep, diagnose the archetype, and write out the first coaching conversation before you have it.

The point: you walk into the 1:1 having already done the diagnostic. You're not figuring out the problem in real time — you're testing a hypothesis you've already built.

## The Standard You're Aiming For

Before this session: *"I need to talk to [rep] about their AI usage."*  
After this session: *"[Rep] is an Over-Reliant / AI-Avoider / AI-Native with [gap]. The first conversation is about [specific behavior]. I'm not coaching the tool — I'm coaching the result."*

The diagnosis is the coaching leverage. Without it, you're giving advice. With it, you're managing.
