# Managed Agents as a Service Business — Playbook

> Framework for selling Claude Managed Agent deployments to non-technical businesses. Directly relevant to Andrew's $15k/month escape velocity goal.

## The Business Model

### Revenue Structure
| Component | Price |
|-----------|-------|
| AI Audit (discovery) | $999 |
| Agent setup | $1,500–$5,000 |
| Monthly maintenance | $500/month |

**Math to $15k/month:**
- 10 clients × $500/mo maintenance = $5,000/mo recurring
- Plus setup fees: 2-3 new clients/month × $3,000 avg = $6,000-9,000/mo
- Plus audits: 3-4/month × $999 = $3,000-4,000/mo
- Blended: 10 active clients puts you in the $10,000-15,000/mo range

### Cost Structure
- Standard Claude API token rates (same as normal Claude usage)
- $0.08 per session-hour for active runtime
- Web search: $10 per 1,000 searches
- Typical agent session (10 min): a few cents
- Heavy client usage: well under $100/month
- **Margin is massive** — $500/mo revenue vs ~$50-100/mo cost per client

## The Playbook

1. **Audit** ($999) — Identify client's biggest time wasters
2. **Build** — Create managed agent that solves their #1 problem
3. **Deploy** — On Anthropic's infrastructure (they handle security, uptime, sandboxing)
4. **Maintain** ($500/mo) — Improve, tune, expand scope

## Sellable Use Cases

### Tier 1: Easy sells (low complexity, high perceived value)
- Email reader/drafter in client's voice
- Weekly competitor briefing generator
- Client onboarding automation (collect docs, send reminders)

### Tier 2: Higher value (moderate complexity)
- Multi-source report generators for executives
- Customer support agent (answer Qs, escalate, log to CRM)
- Document processors (extract data from raw files)

### Tier 3: Premium (high complexity, high stickiness)
- Project management agents (join Asana/Linear, pick up tasks)
- Full content pipeline (notes → blog posts, social, newsletters)
- Custom workflow agents tied to client's specific toolstack

## The Permission System (Why Clients Say Yes)

Two modes, mixable per action:
- **Auto-run** — Agent handles it (internal tools, low risk)
- **Approval required** — Agent pauses before acting (client-facing, sensitive)

**The sales line:** "The agent will draft the email, but it won't send anything without your approval."

This is how you get risk-averse business owners past the "AI will send something embarrassing" objection.
